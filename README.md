# Incubyte Test Task

This project is created using following versions:

```
Node: v18.17.0

Ruby: 3.3.0

Cocoapods: 1.15.2

Java: openjdk 17.0.10 2024-01-16 LTS

```

## For running the application:

Run these commands:

```
npm i && cd ios && pod install && cd ..

```

For Android:

```
npm run android

```

For Ios:

```
npm run ios

```
